サイエンスガイド — 数学編

素数・対数・軌道計算——宇宙の言語としての数学

TARGET: 中学2年〜高校生 SUBJECT: 数学 ⚠ ネタバレあり

数学はこの物語のなかで「宇宙共通の言語」として機能する。グレースとロッキーが最初に交わすのは言葉ではなく、数の列だ。軌道計算も、エネルギー密度の比較も、すべて数学なしには成り立たない。「なぜ数学は宇宙で通じるのか」という問いそのものが、深い哲学的問いを含んでいる。

MATH 01
素数——最初の共通言語
素数 / 整数論 / 数学の普遍性
▶ 中1〜中2レベルから／深みは無限
📖 ストーリー
グレースとロッキーが初めてコミュニケーションを試みるとき、グレースは素数の列を送る。2, 3, 5, 7, 11, 13……ロッキーがその続き（17, 19）を返したとき、「知性との接触」が確認される。共通の文化もなく、共通の言語もない相手と、数だけで通じ合う場面だ。
🔬 数学解説：素数とは何か、なぜ宇宙共通なのか

素数とは「1とその数自身以外に約数を持たない2以上の整数」だ。2, 3, 5, 7, 11, 13, 17, 19, 23……と無限に続く。


素数が「宇宙共通言語」として使える理由：

  • 素数の概念は文化・言語・物理環境に依存しない
  • 「規則性があるが単純な繰り返しではない」パターンは、知性によって識別される
  • 宇宙人と接触した場合、素数列を送ることは「人工的な知性の存在」を示す最も確実な方法のひとつとして、SETI（地球外知性探査）でも真剣に議論されている

素数の個数は無限に存在する
ユークリッドによる証明（背理法、紀元前300年頃）：

「素数が有限個（p₁, p₂, ..., pₙ）しかないと仮定する。
N = (p₁ × p₂ × ... × pₙ) + 1 を考えると、
Nはどの素数でも割り切れない（余りが1になる）。
よってNは新たな素数か、リストにない素数で割れる → 矛盾」

この証明は2300年前に完成しており、今も有効。数学の「不老不死性」の象徴だ。
📐
ユークリッド（紀元前300年頃）

『原論』（Elements）を著し、幾何学と整数論を公理的に体系化。素数が無限に存在することの証明も含まれる。2000年以上にわたって数学の教科書として使われた史上最も長命な数学書。

🔢
カール・フリードリヒ・ガウス（1777〜1855）

「数学の王者」と称される。素数の分布を表す素数定理（π(x) ≈ x/ln x）を予測。整数論・複素数・統計学・測地学など広大な分野で先駆的業績を残した。18歳で正17角形の定規とコンパスによる作図を発見している。

💭 考えてみよう
「双子素数」とは差が2の素数のペア（3と5、11と13、17と19……）だ。双子素数は無限に存在するか？これは現代数学の未解決問題のひとつ。証明できた人は数学者として世界的に有名になれる。

MATH 02
対数スケール——宇宙的な数を扱う道具
対数 / 指数関数 / 天文学的スケール
▶ 中3〜高1レベル
📖 ストーリー
グレースは太陽とタウ・セチ（目的地の恒星）のエネルギー収支、アストロファージの密度、船のエネルギー量など、桁違いに異なる数値を扱い続ける。こうした「宇宙的な数」を比較・計算するには、通常の算術では限界がある。
🔬 数学解説：対数——「桁」を数える道具

太陽の質量は約2×10³⁰ kg、電子の質量は約9×10⁻³¹ kg。この2つを同じ数直線上で比較しようとすると途方もない。そこで対数（logarithm）を使う。


log₁₀(x) は「10を何乗したらxになるか」を返す。

  • log₁₀(100) = 2 → 10² = 100
  • log₁₀(1,000,000) = 6 → 10⁶ = 1,000,000
  • log₁₀(0.001) = -3 → 10⁻³ = 0.001

対数を使うと、何十桁もの差がある数を「数桁の差」として扱える。天文学・地震学（マグニチュード）・音響学（デシベル）・化学（pH）はすべて対数スケールだ。


さらに、指数関数と対数は逆関数の関係にある。細菌の増殖、放射性崩壊、宇宙の膨張——急激な増加・減少はすべて指数関数で記述される。

y = log_a(x) ⟺ a^y = x
a：底（base）。常用対数は a=10、自然対数は a=e≈2.718
x：真数（正の実数）
y：対数の値

重要な性質：
log(A × B) = log(A) + log(B)　← 掛け算が足し算になる
log(Aⁿ) = n × log(A)　← べき乗が掛け算になる

この性質のおかげで、ケプラーやニュートンの時代は対数表を使って天文計算をしていた。
📊
ジョン・ネイピア（1550〜1617）

対数を発明（1614年）。天文学者が何時間もかけていた複雑な掛け算を、対数表を使えば足し算で解けることを示した。ケプラーやガリレオも対数表を愛用した。計算機が発明される300年前の「計算革命」だ。

🧪 対数スケールで宇宙を俯瞰する
  1. 電子（10⁻³⁰ kg）から可観測宇宙（10⁵³ kg）まで、対数スケールで並べてみよう
  2. 地球から月（3.8×10⁸ m）→ 太陽（1.5×10¹¹ m）→ 最近傍の星（4×10¹⁶ m）を log₁₀ で計算すると何桁違うか？
  3. 「光年」が何メートルか計算し、タウ・セチ（約12光年）をメートルで書いてみよう

MATH 03
軌道計算とホーマン遷移
微積分 / ケプラーの法則 / 最適経路問題
▶ 高校〜大学入門レベル（発展）
📖 ストーリー
タウ・セチ星系に到着したグレースは、惑星間を移動するために軌道力学を計算する必要がある。燃料を最小限に抑えながら目的地に到達する「最適軌道」は、ニュートン力学と微積分の結晶だ。
🔬 数学解説：ケプラーの法則と軌道の幾何学

惑星の軌道は円ではなく楕円だ（ケプラーの第1法則）。太陽（または恒星）はその楕円の焦点のひとつにある。


宇宙船が最も燃料効率よく軌道を移動する方法をホーマン遷移軌道という。2回だけエンジンを噴射して、小さな軌道から大きな軌道（または逆）に移る方法だ。


これらの計算には微積分が必要だが、その背後にある考え方は「エネルギーを最小化する経路を見つける」という変分法の問題でもある。現代の宇宙探査機（ボイジャー、カッシーニ）はこれを応用し、複数の惑星の重力を巧みに使う「スイングバイ」で燃料を節約している。

T² = (4π² / GM) × a³
ケプラーの第3法則（調和の法則）
T：公転周期
a：軌道の長半径（楕円の長い方の半径）
G：万有引力定数
M：中心天体の質量

T² ∝ a³（周期の2乗は長半径の3乗に比例）
例：地球（a=1AU, T=1年）→ 火星（a=1.52AU）の T を計算してみよう
🪐
ヨハネス・ケプラー（1571〜1630）

ティコ・ブラーエの膨大な観測データを20年かけて分析し、惑星運動の3法則を発見。「楕円軌道」「面積速度一定」「周期と距離の関係」はいずれも純粋に観測から帰納したものだ。ニュートンはこれを万有引力で理論的に説明した。

∫
ゴットフリート・ライプニッツ / アイザック・ニュートン（17世紀）

微積分を（独立に）発明。ニュートンは惑星運動を解くために微積分を必要とし、ライプニッツは現在使われる記号（dy/dx、∫など）を整備した。2人の間で「先に発明したのはどちらか」をめぐる激しい優先権争いが起きた。
